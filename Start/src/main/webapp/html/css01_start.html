스타일 시트 적용하기

‘중식 대통령’ 이연복 셰프가 ‘라디오스타’에 출연해 세계적인 셰프 고든 램지와 세기의 요리 대결을 펼쳤던 비하인드를 공개한다. 이어 그룹 방탄소년단 진과 절친을 맺게 된 뒷이야기를 전한다고 해 궁금증을 불러 모은다.

오는 28일 밤 10시 30분 방송 예정인 고품격 토크쇼 MBC ‘라디오스타’(기획 강영선/연출 이윤화)는 이연복, 김병현, 노사연, 히밥이 출연하는 ‘수요일은 밥이 좋아’ 특집으로 꾸며진다.

중식 전문가 이연복 셰프는 다양한 방송 프로그램을 통해 뛰어난 중식 요리 실력을 자랑해 시청자들을 사로잡은 스타 셰프다. 2년 만에 ‘라디오스타’에 재방문한 그는 이슈가 됐었던 식당 폐업 소식에 대해 잘못된 내용이 있다며 직접 해명에 나선다.
